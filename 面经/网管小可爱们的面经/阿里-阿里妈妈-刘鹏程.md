用砖头垒墙。假设第一层垒8个，第二层最多只能垒4个，第三层最多只能垒2个，以此类推，那么共有多少种垒法。

思路：dp。设 dp[i] 表示第一层垒 i 个的垒法，初始时 dp[0] = dp[1] = 1，则 dp[i] = dp[0] + dp[1] + ... + dp[i / 2]。

```java
public int count(int n) {
    if (n < 0) return -1;
    int[] dp = new int[n + 1];
    dp[0] = dp[1] = 1;

    for (int i = 2; i<= n; i++) {
        for (int j = 0; j <= i / 2; j++) {
            dp[i] += dp[j];
        }
    }

    return dp[n];
}
```
